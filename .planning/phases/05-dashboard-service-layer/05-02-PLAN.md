<task type="auto" tdd="true">
  <name>Task 1: Add getWeeklyFocus function</name>
  <files>lib/services/dashboard.service.ts</files>
  <behavior>
    - Active journey at day 3 of 14: returns days 3-9 (current + next 6), completedCount for those days, totalCount 7, currentDay is day 3
    - Active journey at day 12 of 14: returns days 12-14 (only 3 remaining), completedCount, totalCount 3, currentDay is day 12
    - Active journey at day 1: returns days 1-7, completedCount 0, totalCount 7, currentDay is day 1
    - No active journey: returns null
    - completedCount only counts days within the returned window that have a check-in
  </behavior>
  <action>
    Add `getWeeklyFocus(userId: string)` to dashboard.service.ts.

    Return type:
    ```typescript
    export interface WeeklyFocus {
      days: Array<{ id: string; title: string | null; position: number; phaseTitle: string; completed: boolean; isCurrent: boolean }>
      completedCount: number
      totalCount: number
      currentDay: { id: string; title: string | null; position: number; phaseTitle: string } | null
    }
    ```

    Implementation (per WEEK-01 and WEEK-03):
    1. Call `getUserActiveJourney(userId)`.
    2. If no active journey or no currentDayId, return null.
    3. Flatten all days from journey.phases (preserving phase title for each day): `journey.phases.flatMap(p => p.days.map(d => ({ ...d, phaseTitle: p.title })))`.
    4. Find currentIndex by matching currentDayId.
    5. Slice a window of up to 7 days starting from currentIndex: `allDays.slice(currentIndex, currentIndex + 7)`.
    6. Get the set of completed day IDs from activeJourney.checkIns: `new Set(activeJourney.checkIns.map(c => c.dayId))`.
    7. Map the window to include `completed: boolean` and `isCurrent: boolean`.
    8. Count completed days within the window only.
    9. Return `{ days, completedCount, totalCount: days.length, currentDay }`.

    Export the WeeklyFocus interface.
  </action>
  <verify>
    <automated>npx tsx -e "import { getWeeklyFocus } from './lib/services/dashboard.service'; console.log(typeof getWeeklyFocus === 'function' ? 'PASS: exported' : 'FAIL')"</automated>
  </verify>
  <done>getWeeklyFocus returns a 7-day window from current position with completion count, or null for users without active journey. WeeklyFocus interface is exported.</done>
</task>
