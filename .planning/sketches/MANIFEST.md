# Sketch Manifest

## Design Direction

The user dashboard is the customer's **entrance to the FamilyMind universe**. The brand promise is **"ro og overblik"** — calm and overview. The current dashboard stacks 9 sections vertically (greeting, personalised subtitle, community pills, daily check-in, weekly focus, completed banner, admin message, courses, recommendations) — that delivers density, not calm.

These sketches explore how the dashboard can embody the brand voice from the first second. Single-focus layouts beat libraries; whitespace and one clear next action beat horizontal scrolling card decks; status moves from in-page sections to small pills or sidebar.

## Reference Points

- **Headspace / Calm** — single-focus daily card, generous whitespace, soft sand palette
- **Apple Health summary card** — one primary metric foregrounded, secondaries quiet
- **Linear inbox empty state** — calm composition that survives empty data
- The platform's own brand language: serif headings (Source Serif), warm sand background, dæmpet nordisk blågrøn primary
